# WorkTracker PRO Landing Page

Это автономный проект лендинга для размещения на хостинге (например, reg.ru).

## Инструкция по установке:

1.  **Скачайте** содержимое этой папки (`/landing-export`).
2.  **Откройте терминал** (командную строку) в этой папке на вашем компьютере.
    *   *Совет для macOS:* Откройте программу **Терминал** (через Spotlight: `Cmd + Space` и введите "Terminal"). Введите `cd ` (с пробелом в конце), а затем просто перетащите папку `landing-export` из Finder прямо в окно Терминала. Нажмите Enter.
3.  **Установите зависимости** (они уже прописаны в файле `package.json`):
    ```bash
    npm install
    ```
4.  **Соберите проект**:
    ```bash
    npm run build
    ```
5.  **Загрузите содержимое папки `dist`** на ваш хостинг (например, через FTP в папку `public_html`).

## Конфигурация:

-   Все ссылки в лендинге настроены на ваше основное приложение: `https://work-tracker-pro.vercel.app/`.
-   Если вы измените адрес приложения, обновите константу `APP_URL` в файле `src/LandingPage.tsx`.
